# הוראות טכניות לבניית עמודי HTML - אתר הכנה לדיון

## עקרונות יסוד

### 1. Mobile First - עיקרון מנחה
העמודים נבנים **קודם למובייל** ואז מותאמים למסך גדול יותר.
- עומרי יקרא מהטלפון בבית המשפט
- כל עיצוב מתחיל מרוחב 320px
- Media queries רק להרחבה (min-width), לא לצמצום

### 2. עיצוב - סגנון Claude Dark
- **רקע כהה** - כמו ממשק Claude
- **כרטיסים מינימליסטיים** - פשוט ונקי
- **ללא אימוג'י** - רק אייקונים מ-Tabler Icons
- **צבעים רכים** - לא צועקים

## רכיבים מיוחדים לפי סוג עמוד

### 1. ציר זמן - Timeline Component
```html
<div class="card">
  <div class="card-header">
    <i class="ti ti-calendar-event"></i>
    11/2016
  </div>
  <div class="highlight-box info">
    <i class="ti ti-building"></i>
    <div>
      <strong>תובעת:</strong> תשלום התקבל ב-20.06.2017
      <div class="quote-source">סעיף 12</div>
    </div>
  </div>
  <div class="highlight-box warning">
    <i class="ti ti-truck"></i>
    <div>
      <strong>נתבעת:</strong> אין התייחסות
    </div>
  </div>
  <div class="highlight-box danger">
    <i class="ti ti-alert-circle"></i>
    <div><strong>סתירה:</strong> איחור של 7 חודשים!</div>
  </div>
</div>
```

### 2. שאלת חקירה - Question Component
